# Zentrix — Content context (Backcom-inspired)

This file translates patterns from [Backcom.io](https://backcom.io/) into **Zentrix** messaging. Use it with `docs/spec.md` when writing copy or new sections.

## What Backcom does well (reference)

| Pattern | How Backcom uses it | Zentrix equivalent |
|--------|----------------------|---------------------|
| **Hero clarity** | One line: hoàn phí Crypto/FX, % cụ thể, CTA mạnh | Headline = trade → rebates → wallet; sub = on-chain, no hidden middlemen |
| **Trust bar** | Sàn hợp tác, trader đã tham gia, $ hoàn lại | LP-02 metrics: volume, active traders, brokers, rebates paid |
| **Problem → solution** | Phí đi về influencer/sàn → Backcom chia lại cho trader | Traditional rebate opaque → Zentrix: Pool + explorer + fixed fees |
| **3 bước** | Đăng ký → Liên kết UID → Giao dịch & nhận hoàn | Connect broker → Trade as usual → Claim to wallet |
| **Bảng so sánh** | Sàn, mã ref, % hoàn (minh bạch) | Spec LP-06B broker listing (future); table = Zentrix vs traditional |
| **FAQ dài** | Giải uy tín, cơ chế, phí ẩn, thời gian hoàn | Spec LP-09 — same intent: AML, MLM, Pool safety |
| **Tone** | Fintech, không hứa lợi nhuận đầu tư; nhấn **phí giao dịch** | Same: rebates from **fees**, not investment returns |

## Voice rules

- **Do:** transparent, verifiable, settlement, broker, rebate, on-chain, KYC/AML, referral from trading fees.
- **Don’t:** generic “AI SEO”, guaranteed profit, vague “boost rankings”.
